        - 1️⃣ What is the difference between var, let, and const?
        Answer:: 

        var : Function-scoped, can be redeclared and updated.

        let : Block-scoped, can be updated but not redeclared in the same scope.

        const : Block-scoped, cannot be updated or redeclared (value stays constant).
      
    - 2️⃣ What is the spread operator (...)?

    Answer:: Spread operator (...) is used to expand or copy elements of an array or object into another array, object, or function arguments.


    - 3️⃣ What is the difference between map(), filter(), and forEach()?
      Answer :: 
      map() → Creates a new array by transforming each element.

      filter() → Creates a new array with elements that pass a condition.

      forEach() → Loops through elements but does not return a new array.
